I'll just paste here the usual simple steps to install the latest firefox 3.6 on your ubuntu:  
```
  
sudo add-apt-repository ppa:mozillateam/firefox-stable  
sudo apt-get update  
sudo apt-get install firefox-3.6  
```
  
  
Have fun ;)
