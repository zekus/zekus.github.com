--- layout: post title: upstream sent too big header while reading response header from upstream tags: - buffer - fastcgi - Linux - nginx - PHP - php-fpm status: publish type: post published: true meta: {} --- if you have this error from nginx with php-fpm and output_buffers = On in your php.ini then try to increment the buffer size in nginx:
```
fastcgi_buffers 8 16k;
fastcgi_buffer_size 32k;
```
