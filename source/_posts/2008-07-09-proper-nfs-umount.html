--- layout: post title: proper nfs umount tags: - bash - Linux - nfs - shell status: publish type: post published: true meta: _edit_last: '1' --- Hi if you try to umount an NFS mountpoint this way ``` root:~> umount /mountpoint ``` and your shell answer this way ``` umount: /mountpoint: device is busy umount: /mountpoint: device is busy ``` then try killing all processes using the mountpoint and update the nfs export fs ``` root:~> fuser -k -m /mountpoint root:~> exportfs -au ``` at this point, you can safely umount ``` root:~> umount /mountpoint ``` peace!
