--- layout: post title: the correct procedure for compiling subversion 1.6.2 tags: - Linux - SVN status: publish type: post published: true meta: {} --- Not sure if this is the complete way of doing it (I'm not telling you which .deb to install), but it worked for me:

Remove some waste from eventual previous compilations:
```
$ rm -rf /usr/loca/lib/libsvn*
$ rm -rf /usr/local/lib/libapr*
$ rm -rf /usr/local/lib/libexpat*
$ rm -rf /usr/local/lib/libneon*
```
Run autogen
```
$ ./autogen.sh  
```
Configure and install (it is important to follow the order)
```
$ ./configure
$ make -j external-all
$ make -j local-all
$ sudo make install
```
That's it. Hope to have helped someone as usual.
