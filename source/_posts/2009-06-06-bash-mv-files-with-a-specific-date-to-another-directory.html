'bash: mv files with a specific date to another directory' tags: - Linux - shell status: publish type: post published: true meta: _edit_last: '1' --- I was searching for a bash command to move files and directories from a folder to another with modified in a specific date for ages on google until I gave up and decided to write it myself: ``` ls -l | awk '$6 == "2009-06-04" { print $8 }' | xargs -t -i mv {} /destination/folder ``` To understand what this line does, you need to know a bit about awk and xargs (try "man xargs" in your shell) Explaination: ls -l : directory listing awk '$6 == "2009-06-04" { print $8 } : take the files and dir with date 2009-06-04 and print the list xargs -t -i mv {} /destination/folder : execute the mv command. {} is the result from the previous command.
