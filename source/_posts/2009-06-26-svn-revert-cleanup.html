--- layout: post title: svn revert cleanup tags: - awk - shell - SVN - xargs status: publish type: post published: true meta: {} --- You have reverted an svn merge and you have a lot of files not tracked (with exclamation mark, basically)?
Clean it up:
``` svn status | awk '$1 == "?"  {print $2}' | xargs -t -i rm -r {} ```
Again, knowing awk and xargs turn helpful!
