--- layout: post title: disable Pure-ftpd logging on syslog tags: - Linux - log - pure-ftpd - shell - ubuntu status: publish type: post published: true meta: _edit_last: '1' --- Pure-ftpd is a bit tricky to configure because do not handle the configuration like any other daemon in Linux so you will not find any /etc/pure-ftpd.conf. According to the pure-ftpd documentation, you should start the daemon already with the options you need. The default Ubuntu installation, use the files in /etc/pure-ftpd/conf/ to set the options for the startup script of pure-ftpd so, to disable the logging, execute the following command in your shell and restart the daemon ``` $ echo "none" > /etc/pure-ftpd/conf/SyslogFacility $ /etc/init.d/pure-ftpd-mysql restart ``` ;)
