--- layout: post title: /media error resolved tags: - Linux status: publish type: post published: true meta: blogger_blog: zekus.blogspot.com blogger_author: zekus blogger_permalink: /2005/07/media-error-resolved.html --- Have you got any problem wit /media directory? cannot automatically mount cd or usb drives? well, your solution is in fstab ( /etc/fstab ).
Check if in fstab that media directory is mounted Read-Only ( ro ). If so, edit it and change ro with rw ( read-write ). Well, now you can umount /media/ and mount it again ( mount /media/ )!!
