--- layout: post title: Multiple rename with find command! tags: - Linux status: publish type: post published: true meta: blogger_blog: zekus.blogspot.com blogger_author: zekus blogger_permalink: /2004/09/multiple-rename-with-find-command.html --- I've done this shell script for multiple rename files not in same folder
(for doin that, simply use rename command) using the combination between mv and find in this way:
for i in $(find / -name *.ext1);
do mv $i $(echo $i | cut -f1 -d.).ext2;
done
in the specific, this change an extension of a file (ext1) in one another (ext2).
feel free to modify it to match your needs.
